Gnosis Safe Contracts
=====================

Install
-------
### Install requirements with npm:

```bash
npm install
```

### Run all tests (requires Node version >=7 for `async/await`):

```bash
truffle compile
truffle test
```

### Deploy

```bash
truffle deploy
```
